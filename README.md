# Zombie Chaos

Um jogo desenvolvido 100% em Java, focado em sobreviver às hordas de zumbis enquanto melhora seu personagem, feito utilizando estrutura de pacotes organizada e lógica de programação pura.

# Sobre o projeto

Este projeto foi criado para a disciplina de Programação Orientada a Objetos. O diferencial é não utilizar engines externas, focando na utilização de Java puro para a renderização e controle de eventos.

O projeto se trata de um jogo com visão de cima, no qual o jogador deve sobreviver às hordas de zumbi, gerenciando sua munição e recebendo dinheiro por matar os zumbis que vão aparecendo pelos cantos da tela. No final de cada horda o jogador poderá comprar melhorias para o personagem numa loja que aparecerá na tela, e ao fechar a loja a próxima horda se inicia, com a dificuldade maior que a anterior.

## Tecnologias utilizadas

**Linguagem**: Java 22

**IDE**: IntelliJ IDEA

**Biblioteca gráfica**: Swing (com eventos de input do AWT)

# Como executar

Para rodar o projeto localmente, siga esses passos:

**1. Clone o repositório:**

```Bash
git clone https://github.com/Brugo24/Zombie_Chaos.git
```

**2. Localize e execute o arquivo joguin.jar**

Zombie_Chaos/out/artifacts/joguin_jar/joguin.jar

# Controles

**WASD:** Movimentação

**Mouse:** Mira

**M1:** Disparo

**M2:** Ataque corpo a corpo
